Acerca del Add-on / DataPack

"Si tocas el bloque, mueres" es un Add-on y DataPack para Minecraft Bedrock y Java Edition respectivamente (Mod de Minecraft a partir de aquí).

Este Mod de Minecraft es un minijuego donde no puedes pisar los una lista seleccionada de bloques, dependiendo del modo de juego que elijas. Si pisas un bloque prohibido, instantaneamente morirás.

Cómo configurar la modificación

La Mod de Minecraft se configura de forma distinta en cada versión. A continuación, explicaré como configurarla en cada versión.

Minecraft Bedrock

Para configurar el Add-on en Minecraft Bedrock, ve a los ajustes de los paquetes de comportamiento y selecciona la opción de "Ajustes" del Add-on.

Cuando presiones el icono "ajustes", se abrirá una ventana con una barra deslizable. Puedes usar esa barra deslizable para cambiar el modo de juego del Add-on (por defecto "crimson nylium" [Necelio carmesí]).

Nota: Para que el Add-on funcione correctamente, necesitas activar dos funciones experimentales (experimentos):

  • Próximas funciones para creadores
  • API Beta




Minecraft Java

El DataPack de Minecraft Java solo se puede configurar durante la partida uArenao Ítems personalizados que recibes cuando apareces en el mundo. Para obtener esos Ítems, la barra de acceso rápido tiene que tener libres las ranuras 9, 8, y 7.

"Prev game mode" (Modo de juego previo): Este objeto te permite cambiar al modo de juego anterior (numéricamente).

"Reset game mode" (Reiniciar el modo de juego): Este objeto te permite reiniciar el modo de juego al primero (numéricamente): "All Blocks" (Todos los bloques).

"Next game mode" (Siguiente modo de juego): Este objeto te permite cambiar al modo de juego siguiente (numéricamente).

Nota: Para una mejor experiencia, deberías de activar el paquete de texturas, el cual esta incluido en las descargas de la versión para Minecraft Java.

Notas:

En esta sección encontrarás notas importantes que deberías de tomar en cuenta si planeas jugar la Mod de Minecraft.

Al aparecer...

Si el modo elegido incluye bloques donde usualmente apareces, como el bloque de pasto o tierra, en la lista de bloques prohibidos, Morirás al instante. Para evitar un bucle infinito de muerte, se creará una plataforma de madera en las coordenadas 0 125 0, y el punto de generación de tu mundo se moverá allí. Esa plataforma está iluminada y tiene una columna de agua para descender.

Paridad...

Hice todo lo posible para que el Add-on y el DataPack funcionen de la misma manera, pero debido a las diferencias entre las versiones, hay cambios menores que no afectan significativamente la experiencia del juego.
