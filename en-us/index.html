About the Add-on / DataPack

"If touch the block, die" is an Add-on and DataPack for Minecraft Bedrock and Java Edition (Minecraft Mod from now).

This Minecraft Mod is a minigame in which you can't step on the selected list of blocks depending on the game mode you selected, and if you step on an unallowed block, you'll instantly die.

How to configure the mod

The Minecraft mod can be configured, but the way to configure it is different on both platforms, below I'll explain how you can configure it for each version.

Minecraft Bedrock

To configure this Add-on for Minecraft Bedrock, you need to go to the settings area for the Add-on, which is on the "behaviour pack" section, and on the "settings" option on the Add-on.

While you click the "settings" icon, a window will open with a slider, on this slider you can change the game mode of the Add-on (crimson nylium by default). Just slide the slider to change the mode.

Note: For the correct Add-on working, you need to enable two experimental features (experiments):

  • Upcoming Creator Features
  • Beta APIs




Minecraft Java

The Minecraft Java DataPack can be configured just in-game with custom items that you receive while spawning. To receive this items you need to have empty the hotbar slots 9, 8, and 7.

Prev game mode: with this item you can change to the game mode before (numerically) the active.

Reset game mode: With this item you can set the game mode to the first one (numerically): "All blocks".

Next game mode: With this item you can change to the game mode after (numerically) the active.

Note: For a better experience, you have to enable the resources pack, which is included on the downloads for Minecraft Java too.

Notes:

On this section you'll se some important notes to consider for while you're playing the Minecraft Mod.

While spawning...

If the choosen mode is one in which the usual blocks where you spawn, like grass, dirt, etc. are listed to kill you, you'll die while you spawn, but for avoiding an infinite loop of death, on the 0 125 0 coordinate, a wooden platform is created, and your spawn point is moved to that platform. That platform is iluminated and has a column of water for descending.

Parity...

I did my best for making the Add-on and DataPack work as the same, but it isn't completely possible as both versions are different, so, some little changes exists there, but they almost don't affect the game experience.
