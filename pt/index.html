Sobre o complemento / Pacote do dados

"Se toque no bloco, morra" é um complemento e Pacote do Dados para Minecraft Bedrock e Java Edition (Minecraft Modificação a partir de agora).

Esta Minecraft Modificação é um minijogo onde você não pode pisar em uma lista selecionada de blocos, Dependendo do modo de jogo que você escolher. Se você pisar em um bloco proibido, você morrerá imediatamente.

Como configurar o modificação

A Minecraft modificação pode ser configurada de maneira diferente em cada plataforma. Abaixo, explicarei como configurá -lo para cada versão.

Minecraft Bedrock

Para configurar este complemento para o Minecraft Bedrock, vá para a área de configurações sob o seção "Comportamento Pack" e selecione a opção "Configurações" para o complemento.

Quando você clica no ícone "Configurações", uma janela será aberta com um controle deslizante. Você pode usar esse controle deslizante para alterar o modo de jogo do complemento (Nicélio distorcido por padrão).

Nota: Para que o complemento funcione corretamente, você precisa ativar do jogabilidade experimental (experimentos):

  • Próximos Recursos de Criador
  • APIs beta




Minecraft Java

Minecraft Java Pacote do Dados só pode ser configurado no jogo usando itens personalizados que você recebe Após a desova. Para obter esses itens, os slots de Barra rápida 9, 8 e 7 devem estar vazios.

"Prev game mode" (Modo de jogo anterior): Este item permite que você mude para o modo de jogo anterior (numericamente).

"Reset game mode" (Redefinir o modo de jogo): Este item redefine o modo de jogo para o primeiro (numericamente): "All Blocks" (Todos os blocos).

"Next game mode" (Próximo modo de jogo): Este item permite que você mude para o próximo modo de jogo (numericamente).

Note: For a better experience, you should enable the resource pack, which is included in the downloads for Minecraft Java.

Notes:

In this section, you'll find important notes to consider while playing the Minecraft Modificação.

While spawning...

If the chosen mode includes blocks where you usually spawn, like grass or dirt, as kill blocks, you'll die upon spawning. To avoid an infinite loop of death, a wooden platform will be created at coordinates 0 125 0, and your spawn point will be moved there. That platform is iluminated and has a column of water for descending.

Parity...

I did my best to make the Add-on and Pacote do dados work the same, but due to differences between the versions, there are minor changes that don't significantly affect the game experience.
